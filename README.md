# Étude mondiale de l'accès à l'eau potable

## Contexte

Ce projet a été réalisé dans le cadre de la formation Data Analyst d'OpenClassrooms.

L'objectif était d'identifier les pays et régions prioritaires pour le développement de l'accès à l'eau potable à partir de données internationales.

## Objectifs

* Analyser les indicateurs liés à l'accès à l'eau potable.
* Identifier les zones prioritaires pour les investissements.
* Construire des visualisations facilitant la prise de décision.
* Présenter les résultats sous forme de Data Storytelling.

## Outils utilisés

* Tableau
* Data Visualization
* Business Intelligence

## Compétences développées

* Data Storytelling
* Tableau Dashboard
* KPI Analysis
* Data Visualization
* Business Analysis

## Résultats

* Identification des pays prioritaires.
* Construction de tableaux de bord interactifs.
* Présentation de recommandations destinées aux décideurs.
* Communication visuelle des résultats à travers une Data Story.
